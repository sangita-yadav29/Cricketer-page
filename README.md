# 🏏 KL Rahul Profile Website

A simple and responsive cricket profile website built using HTML, CSS, and Bootstrap. The website showcases information about Indian cricketer KL Rahul, his IPL records, and video highlights.

## 🚀 Features

- Player profile section
- IPL records card
- Video highlights section
- Responsive layout using Bootstrap
- Clean and attractive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 4

## 📂 Project Structure

```text
├── index.html
├── style.css
└── README.md
```
